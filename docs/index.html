財經日曆桌布

目前版本 v0.0 · Windows 10/11（ARM64 / x64）

總經日曆、台股事件、即時行情條，全在你的桌面上。深色毛玻璃風格，資料每 6 小時自動更新，原生 WebView2 渲染、穩定不崩。

⬇ 下載最新版 GitHub 原始碼
📦 下載解壓後，執行 setup.bat 就一鍵裝好環境 — 自動安裝 Python 與 Lively、建立自動更新；最後在 Lively 拖一下 html 設成桌布即可。

特色

📅

總經日曆

Forex Factory 美國・歐元區・日本中高重要性事件，含預測/前值，跟隨系統時區顯示。

📈

台股事件

法說會・股東會預告＋處置股（當日制、遇休市自動順延），資料來自 TWSE／TPEx。

💹

即時行情條

13 檔跑馬燈：加權・台積電・道瓊・費半・SOFR…，紅漲綠跌。

🪟

可調毛玻璃

深色毛玻璃質感，介面縮放／不透明度／模糊／強調色皆可在 Lively 屬性面板即時調。

一鍵安裝

  1. 下載並解壓本專案（上方「下載最新版」）。
  2. 以系統管理員身分執行 setup.bat — 自動裝 Python 與 Lively、建立每 6 小時自動更新排程、抓一次最新資料。
  3. 開啟 Lively，把 finance-calendar.html 拖進去設成桌布 — 完成 ✅ 桌面即顯示最新資料。
